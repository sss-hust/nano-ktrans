# 开发日志 2026-04-14

## 完成的工作

- 将仓库版本提升到 `v0.2.0`
- 同步更新 README、PIM 指南和知识库中的当前状态描述
- 准备把现有阶段性成果收敛成明确的 git 版本点

## 当前结论

- `cpu` 和 `cuda_cpu_offload` 已经打通
- `cuda_pim_shadow` 已经接入主推理链路，但还不是 DPU 数值执行
- UPMEM 真实硬件 benchmark 已经拿到多 rank 结果
- 当前最关键的技术缺口是“真实 PIM expert backend”，不是基线 bring-up

## 当前目标理解

- 项目不是单纯把模型跑起来，而是把 Hybrid MoE 推理链路拆成清晰的 GPU / CPU / PIM 分层执行框架
- 短期目标是让 PIM 成为 cold experts 的真实执行 tier
- 中期目标是对比 CPU offload 和 PIM offload 的端到端收益，并沉淀成可复现 benchmark
